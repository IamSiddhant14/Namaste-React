* HMR : Hot Module Reloading , means auto reload on any change

* File Watcher Algorithm(c++) : This keep a check on the file for any change

* Parser minifies the code

* Bundles things up

* Cleaning our code

* Super fast build Algorithm

* Image optimization

* Caching while devlopment

* Compatable with the older version of browser

* Zero config Bundler
